## 正文（直接复制粘贴）

用了半年 Typeless，最后还是自己写了个平替 🛠️

先说说 Typeless 让我受不了的几个点：

❌ **要钱，还不便宜**
月订阅收费，不用也得交钱

❌ **隐私问题**
每次说话都上传到国外服务器，你的声音数据去哪了？没人知道

❌ **中文识别一言难尽**
Whisper 对中文并不是最优解，经常识别出奇怪的错别字

❌ **没有 LLM 后处理**
识别完就完了，口语转书面语？自己想办法

---

所以我写了 VERBATIM，一个完全开源的 Mac 语音输入工具 ✨

✅ **本地 AI，完全离线**
内置 FunASR 本地模型，说话的声音永远不离开你的电脑
（不联网也能用！）

✅ **中文识别准到离谱**
专门调了 FunASR paraformer-zh 模型，还支持讯飞、SiliconFlow
中文普通话识别率吊打 Whisper

✅ **LLM 智能后处理**
说完话自动帮你：
- 逻辑优化：口语 → 书面语
- Code 模式：语音 → 任务清单
- 自定义提示词：想要啥效果就要啥效果
（支持 DeepSeek / 通义千问 / 豆包）

✅ **完全免费开源 MIT 协议**
代码全在 GitHub，想怎么改就怎么改
再也不用担心哪天涨价或者跑路

✅ **极简操作**
按住快捷键 → 说话 → 松开 → 自动粘贴到光标处
三步走，0 额外操作

---

现在还在持续迭代中，有几个功能还想做：
□ Windows / Linux 支持
□ 更多 ASR 提供者
□ 更好的错误提示

**有 Mac 的朋友可以去 GitHub 搜 VERBATIM 试试**
也欢迎来提 Issue，告诉我你最想要的功能是什么👇
